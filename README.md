## 大頭家電商平台需求分析描述:
#### 1.	新使用者必須點擊註冊帳號及密碼，已註冊者可直接登入。
#### 2.	點擊登入，輸入帳號密碼。
#### 3.	進入產品頁面即可點選商品購買。
#### 4.	確認送出訂單後也可以刪除已訂購的商品。
